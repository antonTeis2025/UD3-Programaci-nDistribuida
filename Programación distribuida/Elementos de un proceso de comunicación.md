1. **Mensaje**: Información a intercambiar
2. **Emisor**: Aplicación que manda el mensaje
3. **Receptor**: Aplicación que recibe el mensaje
4. **Paquete**: Unidad básica de información que intercambian dos dispositivos. Un mensaje se puede dividir en varios paquetes.
5. **Canal de comunicación**: Medio por el que se transmiten los paquetes. Conecta el emisor y el receptor.
6. **Protocolo de comunicación**: Determina las reglas que tiene que seguir la comunicación (formato, secuencia de paquetes).
